# WARP+ Keys Collector

a simple tool that automatically collects 15 WARP+ Keys once every hour.<br>
یک ابزار ساده که به‌طور خودکار هر یک‌ساعت یکبار ۱۵ عدد کلید وارپ‌پلاس را جمع‌آوری می‌کند.
